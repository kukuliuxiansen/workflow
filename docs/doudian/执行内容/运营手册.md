## 二、推流逻辑与运营策略

### 2.1 抖店推流机制

抖店的流量分配主要基于以下因素：

1. **商品质量分**：优秀商品获得更多曝光
2. **营销活动**：单品直降、限时抢购等活动可获得专属流量入口
3. **搜索优化**：商品标题、属性完整性影响搜索排名
4. **用户行为**：点击率、转化率决定流量质量评分

### 2.3 高产出运营操作

#### 免费推流方法

1. **看后搜配置（最高优先级）**
   - 路径：首页 → 搜索运营 → 非自营视频 → 批量配置
   - 效果：商品曝光人数提升50%-80%
   - 操作：批量配置视频的看后搜词和承接商品
   - 频率：每日检查并配置新增视频
   - 注意：非自己发的视频有93个待配置，流失85%商品曝光

2. **单品直降/限时抢购**
   - 路径：营销工具 → 单品直降
   - 效果：商品曝光平均提升14%，转化率提升7.8%
   - 要求：设置促销价（需低于近期最低价）
   - 无需审核，立即生效

3. **商品属性优化**
   - 路径：商品管理 → 批量属性优化
   - 效果：完善商品属性可提升搜索排名
   - 操作：补充材质、风格、适用对象等属性

4. **商机中心铺货**
   - 路径：商机中心 → 为你推荐
   - 效果：选择平台推荐的热销类目，获得流量扶持

5. **优惠券工具**
   - 路径：营销工具 → 优惠券
   - 类型：惊喜券、店铺券、商品券

6. **新品起量投放**
   - 路径：首页 → 新品起量 → 一键投放
   - 效果：首单流量补贴1000次/天
   - 同行投放后GMV平均提升约33%

### 2.3 定价策略

- SKU成本低于20元：运费模板选择"低固引2元"
- SKU成本20元及以上：运费模板选择"包邮0.1不含偏远地区"
- 售价公式：成本价 + 邮费 + 20%利润

## 三、商品管理规范

### 3.1 标题规范

- 所有商品标题以"屿泮"开头
- 使用AI生成优化标题（编辑页面有AI标识）
- 字数限制：15-60字符（8-30汉字）

### 3.2 图片要求

- 主图：1:1比例
- 3:4主图：从1:1智能剪裁
- 详情图：删除厂家图、推广图等无关内容

### 3.3 商品选品原则

- 避免食品类（需要资质）
- 避免重复商品
- 选择多个不同小类目
- 优先选择搜索热度高、成交增速快的类目
